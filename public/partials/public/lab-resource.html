漏洞相关

  • sqlmap
    sqlmap是一款非常强大的开源sql自动化注入工具，可以用来检测和利用sql注入漏洞。它由Python语言开发而成，因此运行需要安装python环境。
    sqlmap使用说明: sqlmap.pdf
    sqlmap工具下载: sqlmap.zip
  • ZAP
    Zed Attack Proxy 简写为 ZAP,是一个简单易用的渗透测试工具,是发现 Web 应用中的漏洞的利器,更是渗透测试爱好者的好东西。
    ZAP使用说明: ZAP.pdf
    ZAP官方指南: ZAP官方指南.pdf
    ZAP软件下载: ZAP_WEEKLY_D-2016-09-26.zip
  • w3af
    w3af 是一个 Web 应用程序攻击和检查框架.该项目已超过 130 个插件,其中包括检查网站爬虫,SQL 注入(SQL Injection),跨站(XSS), 本地文件包含(LFI),远程文件包含(RFI)等.该项目的目标是要建立一个框架,以寻找和开发 Web 应用安全漏洞,所以很容易使用和扩展.
    源码下载: w3af-master.zip
    w3af官网: w3af.org
    github: github.com/andresriancho/w3af
  • DVWA
    漏洞演示系统DVWA(Damn Vulnerable Web Application)
    DVWA指南: DVWA指南.pdf
    DVWA下载: DVWA.zip
  • bwapp
    buggy web Application 这是一个集成了各种常见漏洞和最新漏洞的开源 Web 应用程序,目的是帮助 网络安全爱好者、开发人员和学生发现并防止网络漏洞。包含了超过 100 种漏洞,涵盖了所有主要的已知 Web 漏洞,包括 OWASP Top10 安全风险,最重要的是已经包含了 OpenSSL 和 ShellShock 漏洞。
    bwapp说明文档: bwapp文档.pdf
    bWAPP_latest软件包: bWAPP_latest.zip
